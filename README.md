<!-- GETTING STARTED -->
## Getting Started

### Installation

1. Clone the repo
    ```sh
    $ git clone https://github.com/Nochii1/flowers-ecommerce-site.git && cd flowers-ecommerce-site
    ```
2. Install NPM packages
    ```sh
    $ npm install
    ```

<!-- USAGE EXAMPLES -->
### Running project
* development
    ```sh
    $ npm run start:dev
    ```
* production
    ```sh
    $ npm run build
    #and then
    $ npm run start:prod
    ```
